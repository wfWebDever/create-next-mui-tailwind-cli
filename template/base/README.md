## Technology Architecture

Nextjs15 + Tailwindcss + MUI

## Setup

- **Clone**

  ```bash
  git clone XXX
  cd  XXX
  ```

- **Install Dependencies**

  ```bash
  yarn install
  ```

- **Environment Configuration**
  Create `.env.local` file in root directory:

  ```bash
  cp .env .env.local
  ```

  Modify `.env.local` file to your own environment

  ```bash
  # STAGING
  STAGING=development
  ```

- **Run Development Server**

  ```bash
  yarn dev
  ```
